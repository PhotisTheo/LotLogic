Trigger Market Values Computation

What will this do?

This will start a background task that:

  • Processes all towns and parcels in the database
  • Computes market values using the hybrid valuation engine
  • Uses 365 days of sale history for comparable selection
  • Selects up to 5 comparable properties per parcel
  • Prevents commercial/residential mixing (bug fix applied)
Important Notes
  • This process may take 1-6 hours depending on data size
  • The task runs in the background via Celery worker
  • You can close this page after starting - the task will continue
  • Monitor progress in the Celery worker logs
Configuration:
Lookback Period: 365 days (1 year)
Target Comps: 5 per parcel
Batch Size: 500 records per database write
Scope: All towns and parcels
